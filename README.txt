慈悲山汽车旅馆官网使用说明

1. 网站现在是多页面结构，请从 index.html 打开主页。
2. 导航栏会分别跳转至 about.html、rooms.html、staff.html、reviews.html 和 contact.html。
3. 请让所有 HTML、styles.css 与 imageslogo.jpg 始终位于同一个文件夹。
4. 上传到 GitHub 时，请一次性上传整个 mercy-hill-motel 文件夹中的内容。
5. 后续制作 ARG 隐藏页面时，可以新建 HTML 文件而不把链接放入导航栏。
6. 当前视觉沿用参考图风格：砖红色页眉、深绿色导航栏、米白背景与白色圆角内容卡片。

当前客房安排：

- 双人标准间：201–216，共 16 间，可住 32 人。
